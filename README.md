## EyeWitness

A Python and Kivy-built app for streamlined incident reporting. 

## Dependencies:
- `pip install kivy[base]`
- `pip install kivymd`
- `pip install kivy-garden`
- `pip install kivy-garden.graph`
- `pip install mysql-connector`
- `pip install mysql-connector-python`
